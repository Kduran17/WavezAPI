What is 'Wavez'?

Wavez is an online platform focused on providing real-time, daily wave forecasts for Los Angeles' beaches. Surfers can see view information on wave height, wave direction, and wave period. Think of it as a last minute game changer for beginner/intermediate surfers. Whether you forgot to plan out the surfing trip correctly or want to learn how to surf, Wavez helps you find the best waves to ride.

How does 'Wavez' actually work?

It's simple. No, really, it is. Simply choose which part of Los Angeles you're in, select the desired beach you want to surf at, and Wavez does the rest. Each beach has specific coordinates attached to it, and whenever it's selected, the 'Meteo' API is called and real-time information on surfing variables is delivered. Get off the couch and go crush those waves.

Is it free?

Yes, it's free. Wavez provides live information that is otherwise accesible to the general, surfing public. We just make it simple. However, rumor say's you'll look better surfing with some of our custom Wavez boards.
